# 📂 Projects Included

## 🌐 Task 1 – AI Language Translator

An AI-powered multilingual language translation application built with **Python** and **Streamlit**. The application translates text between multiple languages, supports text-to-speech, and provides an intuitive user interface for seamless communication.

### Highlights

- Translate text into multiple languages
- Text-to-Speech (TTS)
- Copy translated text
- Language swap feature
- Character & word counter
- Modern Streamlit UI

### Tech Stack

- Python
- Streamlit
- deep-translator
- gTTS
- Pyperclip

## 💬 Task 2 – FAQ Chatbot

A Natural Language Processing (NLP) based chatbot that answers frequently asked questions by finding the most relevant response from predefined datasets using text similarity techniques.

### Highlights

- Multiple FAQ datasets
- NLP-based preprocessing
- TF-IDF Vectorization
- Cosine Similarity Matching
- Interactive Streamlit Interface
- Fast and accurate responses

### Tech Stack

- Python
- Streamlit
- NLTK
- Scikit-learn
- Pandas

## 🎥 Task 3 – Object Detection And Tracking System

An AI-powered real-time object detection and tracking system capable of detecting, classifying, and tracking multiple objects from webcam and video streams using state-of-the-art computer vision models.

### Highlights

- Real-time object detection
- Multi-object tracking
- Object counting
- Detection logging
- CSV export
- Session analytics
- Screenshot capture
- Processed video saving

### Tech Stack

- Python

- YOLOv8
- ByteTrack
- NumPy
